# Student Manager

Application for e-commerse

Built using Express.

## Getting started

1. Install dependencies

`npm install`

2. Run application

`npm start`

## Install GIT-LFS for large .jpg files

1. Go to https://git-lfs.com/ download Git LFS and install

2. In terminal run: `git lfs install`

3. Select the files you want Git LFS to manage. run: `git lfs track "*.jpg"`

## Install multer to use images for Admin log in

1. npm install multer 

### Reset Db when working or updating

1. localhost:3000/reset-db

### Git commit (create a new branch and push branch to Github)

1. `git checkout main`
2. `git pull`
3. `git checkout -b your-branch`

### When your done and want to push to Github

4. `git add .`
5. `git commit -m "..."`
6. `git push -u origin your-branch`
